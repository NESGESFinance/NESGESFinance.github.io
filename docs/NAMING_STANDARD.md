# NESGESFinance Naming Standard v2.2

## Nombre institucional principal

NESGESFinance Ecosystem S.A.S. & LLC

## Entidad Ecuador

NESGESFINANCE ECOSYSTEM S.A.S. Sociedad de Beneficio e Interés Colectivo

## Entidad Estados Unidos

NESGESFinance Ecosystem S.A.S. LLC

## Activo utility

NGF•BTC•AM

## Variantes técnicas permitidas

- NGF•BTC•AM: forma visual oficial
- NGF·BTC·AM: forma documental alternativa
- NGF-BTC-AM: forma compatible con ciertos sistemas ASCII, exploradores o JSON

## Uso correcto

NGF•BTC•AM debe describirse como:

- Rune sobre Bitcoin
- utility token
- activo de acceso
- activo de coordinación ecosistémica
- activo de identidad funcional

## Uso prohibido

NGF•BTC•AM no debe describirse como:

- acción
- bono
- deuda
- título de propiedad
- dividendo
- promesa de retorno
- security
- participación directa en inmuebles
- garantía financiera

## Ordinales de Proyecto

Los Ordinales de Proyecto deben describirse como instrumentos independientes, vinculados a documentación, metadata, contratos y proyectos específicos.

Cuando representen derechos económicos, deben tratarse como instrumentos sujetos a revisión legal, KYC/AML y posible regulación de valores.
